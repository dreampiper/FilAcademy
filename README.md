<h2 align='center'>Getting started</h2>

First, you need to open a `Terminal` and clone this repo by using :

```bash
git clone https://github.com/dreampiper/FilAcademy
```

Enter to the folder :

```bash
cd FilAcademy
```

And install dependencies

```bash
npm install
```

Finally, run in dev :

```bash
npm run dev
```
